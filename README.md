# knotable-props-mailer aka props.knote.com app

This repository is for the "knotable-props" application

This app was built by Evgeny and maintained recently by Yaroslav

The app is hosted at http://props.knote.com and routes through mailgun.com

Issues and how-tos for the Knotable, Knote, and Knotel staff users are at https://knote.com/p/P8c0nW/How-to-write-emails-using-props-10th-installment#

Users of the props app need to be part of this group as members https://github.com/orgs/Knotable/teams/email-props/members


## The app users membership.

To add a new GitHub user to the app without purchasing a new seat per user for Knotable organization, you should add the user as an outside collaborator to this PUBLIC [repository](https://github.com/Knotable/props-mailer-access).
Removing the user from that repository will also remove the user's app access.

Users of the props app need to be part of this group as members https://github.com/orgs/Knotable/teams/email-props/members
